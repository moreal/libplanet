Type Description
IEnumerable<(T1 Item1, T2 Item2, T3 Item3)<KeyBytes, Bencodex.Types.IValue, Bencodex.Types.IValue>>

Tuple<T1,T2,T3>s consists of different nodes. The first element is different values' key. The second element is the value at the given origin trie. And the third element is the value at the given other trie but it is null if it doesn't exist in other trie.
