Type Name Description
PolicyActionsGetter beginBlockActionsGetter

A delegator to get policy block actions to evaluate at the beginning for each IPreEvaluationBlock that gets evaluated. Note the order of the returned list determines the execution order.

A delegator to get policy block actions to evaluate at the end for each IPreEvaluationBlock that gets evaluated. Note the order of the returned list determines the execution order.
